claude-desktop-buddy — CYD flasher

Flash the firmware to your Cheap Yellow Display (ESP32-2432S028R) directly from this page. No downloads, no command line.

  1. Plug your CYD into your computer with a USB-C cable.
  2. Click Connect below and pick the serial port that appears (it'll be named something like CH340 or CP210x).
  3. Click Install. Takes about 30 seconds.

Your browser doesn't support the Web Serial API. Use Chrome, Edge, or Opera on a desktop computer. Firefox, Safari, and mobile browsers can't talk to USB devices from a web page.

After flashing

On first boot the device will ask you to calibrate the touchscreen — tap each of the four corner targets. Then pair it with Claude:

  • Claude Desktop (macOS / Windows): enable developer mode (Help → Troubleshooting), then Developer → Open Hardware Buddy → Connect.
  • Claude Code (Linux / macOS / Windows): install the claude-hardware-buddy plugin.
